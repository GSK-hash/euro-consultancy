## üõ†Ô∏è How to Customize

If you want to edit this website later, here are the quick steps:

### 1. Change Contact Email
To receive inquiries in your inbox:
1.  Open `index.html`.
2.  Search for `<form action="...">`.
3.  Replace `your-email@gmail.com` with your actual email address.

### 2. Update Phone Number
1.  Open `index.html`.
2.  Search for `98765 43210`.
3.  Replace it in **three places**:
    * The Text (What people see).
    * The `href="tel:..."` (For clicking to call).
    * The `href="wa.me/..."` (For WhatsApp).

### 3. Change Images
* **Logo:** Upload a new file named `logo.png` to replace the current one.
* **Office Background:** Upload a new file named `office.jpg`.
